## 🚀 Featured Projects

### 🏛️ RTS — PMRDA Government Portal &nbsp;![Live](https://img.shields.io/badge/●_LIVE-00C851?style=flat-square)

> A citizen-facing **Real Time Services** portal serving thousands of residents in the Pune Metropolitan Region. Handles multiple civic services, fee collection, challan generation, fire NOC approvals, and online payments.

**Key contributions:**
- ⚙️ Built a **penalty calculation engine** — 18% per annum after 30-day grace period
- 💳 Integrated **BillDesk payment gateway** for secure citizen fee collection
- 🔒 Implemented **AES/GCM encryption** for receipt numbers in URL parameters
- 🗄️ Designed `tbl_challan` schema changes and Hibernate named queries
- 🌐 Wired **JSP frontend flows** for 10+ citizen services with zero breaking changes
